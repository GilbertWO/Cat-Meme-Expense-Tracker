# 🐱 Meow Expense Tracker

A smart and simple personal expense tracking mobile application built using Flutter.
This app helps users manage their daily expenses, visualize spending habits, and gain useful financial insights.

---

## 📱 Features

* ➕ Add and manage daily expenses
* 📂 Categorize expenses (Food, Transport, Shopping, etc.)
* 📊 Visualize spending with charts
* 📅 Track expenses by date
* 🧠 Basic spending insights and summaries
* 🗑️ Edit and delete expenses

---

## 🛠️ Tech Stack

* **Frontend:** Flutter
* **Language:** Dart
* **Database:** SQLite / Firebase (depending on your implementation)
* **Libraries:**

    * fl_chart (for data visualization)
    * sqflite / firebase packages

## 🚀 Getting Started

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

### Installation

```bash
git clone https://github.com/GilbertWO/Meow-Expense-Tracker.git
cd Meow-Expense-Tracker
flutter pub get
flutter run
```
